Modal

Son los pop-ups que algunas páginas muestran al realizar alguna acción


Título del Modal

Aquí vamos a poner el contenido del modal, a manera de incepción este es el código de este modal.





<!-- Botón que activa el Modal -->

<button class="btn btn-primary" data-toggle="modal" data-target="#myModal">Da click aquí para ver un ejemplo</button>


<!-- Modal -->

<div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="exampleModalLabel" aria-hidden="true">
  <div class="modal-dialog" role="document">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="myModal">Título del Modal</h5>
        <button type="button" class="close" data-dismiss="modal" aria-label="Cerrar"></button>
      </div>
      <div class="modal-body">
        <h4>Aquí vamos a poner el contenido del modal, a manera de incepción este es el código de este modal.</h4>
           <!-- Aquí va el contenido -->
           ...
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-dismiss="modal">Close</button>
      </div>
    </div>
  </div>
</div>

<a href="#" data-toggle="tooltip" title="Hooray!">Hover over me</a>

Dropdown

Es un botón que despliega más opciones en un menú, se utiliza principalmente para navegación


  • Modals
  • Tooltips

   Da click al botón y a las opciones para navegar en esta página

Popover

Se puede entender como un pop-up más avanzado que el tooltip que utilizamos para mostrar información cuando el usuario da click a un elemento


Dame click para ver un popover

<!-- Este es el botón que activa el popover, usamos un id y el campo data-toggle para luego leerlo con jQuery -->

<!-- el campo 'data-placement' indica donde aparecerá, 'title' el título del mismo y 'data-content' el contenido a mostrar -->

<a tabindex="0" id="popover" class="btn btn-lg btn-primary" role="button" data-toggle="popover" data-trigger="focus" data-placement="bottom" title="Ejemplo de popover" data-content="Yo soy un popover y desaparezco cuando haces click en cualquier otro lugar">Dame click para ver un popover</a>

Funciones de Bootstrap con JS

Te invito a explorar más funciones que combinan Bootstrap con jQuery y JS, la documentación es muy completa y lograrás entender como aplicarlos en tus proyectos facilmente


Documentación de Bootstrap-JS

Durante todo el contenido de esta página utilizamos la función Collapse que nos permitió hacer más dinámicas las secciones con código extenso, este tipo de funcionalidad es la que tú tienes que buscar implementar en tu aplicación para mejorar la experiencia del usuario, combinar estas interacciones con la experiencia es lo que nos permite conformar el UX/UI de tus proyectos.




 <!-- Este es el botón que activa y desactiva el elemento a colapsar -->
 <button class="btn" data-toggle="collapse" data-target="#demo">Collapsible</button>

 <!-- Este es el div que mostraremos y ocultaremos, por default está escondido al inicio -->
 <div class="well">
  Yo soy un texto con código oculto para tu comodidad, y así me generan:
  <code id="demo" class="collapse" >
   Contenido...
  </code>
 </div>

 <!-- Esta función, y muchas otras, también necesitan agregar código de JS antes de cerrar la etiqueta body, la función que usamos para ocultar todos los '.collapse' al inicio es la siguiente: -->

  <script type="text/javascript">
   // Estas funciones las ejecutamos cuando carga nuestra aplicación
   $(function() {

    // Usamos la clase junto a la función collapse para mejorar tu experiencia
     $('.collapse').collapse('hide');

   });
  </script>
